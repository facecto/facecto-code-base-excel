# Quick start
## Step 1: Set up pom.xml to add dependencies
```
<dependency>
  <groupId>com.facecto.code</groupId>.
  <artifactId>facecto-code-base-excel</artifactId
  <version>1.0.0</version>.
</dependency>.
```
